# Tempo64 marketing site

Static site for [Tempo64](https://apps.apple.com/us/app/tempo64/id6764440731) — a chess-to-vertical-short iOS app. Served via GitHub Pages at https://chumomega.github.io/tempo64-webpage/.

## Contents

- `index.html` — landing page (overview + feature list + App Store link)
- `privacy.html` — privacy policy referenced from the iOS app and App Store Connect
- `style.css` — shared styles
- `screenshots/` — App Store screenshots + OG cover image
- `app-store-*.txt` — App Store Connect metadata (description, what's new, promo text)
- `icon.png` — small marker icon used by the site

Source assets are generated by `tools/generate_marketing.py` in the (private) iOS repo and synced into this repo when they change.
